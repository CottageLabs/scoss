(static) progressCommitted(component) → {Object}

Function which works out how much progress has been made towards the Service Provider's target in terms of committed funds. This takes the target from the service_provider resource, and the total committed from the total_committed aggregation on the master sheet resource. It returns a percentage up to 100%, which is capped, even if more is committed. It also returns the amount and the target as "x" and "y" respectively, so you can say Z% (X of Y).

(static) topDonorsDFClosure(params) → {Array}

Closure which returns a Data Function which produces source data for the "Top Donors/Members" chart. The returned function fills a single data series with data from the funders and the nested total_committed aggregations on the master_aggregations resource. Funders are sorted according to their contribution. The number of funders returned is limited to the number supplied to this closure, and the limit is applied after sorting. Resulting data series is of the form 
[
 {
     key: "Total Funding",
     values: [
     {
         label: "Funder",
         value: number
     }
     ]
  }
]
